# Oasys Results

Put synthesized netlists and Oasys reports under one directory per config:

```text
results/
├── step1_mode0/
├── step1_mode1/
├── step1_mode4/
├── step2_mode1/
├── step2_mode4/
├── step3_mode0/
├── step3_mode1/
└── step3_mode4/
```

Use the directory name that matches the config file name without `_config.tcl`.
