## Overview

디지털회로설계및언어 프로젝트 2.

프로젝트 1에서 설계한 Simple CPU를 확장하여, Simple CPU가 MMIO 방식으로 제어하는 NPU-style int8 GEMM co-processor / accelerator 구현했다.

| 항목           | 내용                   |
| -------------- | ---------------------- |
| **보드** | Zybo Z7-20 (Zynq-7000) |
| **언어** | Verilog / Python / C++ |
| **클럭** | 125 MHz →             |
